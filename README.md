# 🚀 Animated Resume Portfolio

An interactive and visually engaging portfolio website built to showcase my skills, projects, experience, and achievements through smooth animations and modern web technologies.

🔗 **Live Demo:**  
[View Animated Resume](https://animated-resume-2kss-d5k3nyt3y-gautamjay13s-projects.vercel.app/)

## ✨ Features

- 🎨 Modern and responsive UI
- ⚡ Smooth GSAP-powered animations
- 🌐 Interactive 3D elements with Three.js
- 📱 Mobile-friendly design
- 🚀 Fast performance and optimized assets
- 💼 Professional resume and portfolio showcase
- 🎯 Clean and scalable code architecture

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/gautamjay13/animated-resume.git
```

### Navigate to the project

```bash
cd animated-resume
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
